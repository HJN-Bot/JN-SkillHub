# B2B Enterprise Meeting Plan Template

## Meeting Purpose
Align priorities, resources, development packages, and next two-week deliverables.

## Recommended Compression
Do not open too many meetings by default. Start with one sync and split only when necessary.

### 1. Team Alignment Meeting
Duration: 45–60 min

Agenda:
1. Confirm reporting milestones
2. Confirm project priority
3. Confirm user needs alignment plan
4. Confirm development architecture/work package owners
5. Confirm input material needs
6. Confirm ROI owner and assumptions
7. Assign next two-week tasks

Output:
- priority decision
- owner/due date list
- required follow-up meetings
- decision/blocker list

### 2. User Needs Alignment Follow-up
Only if requirements are not clear.

Output:
- User Requirements Definition
- must-have / should-have / could-have
- acceptance owner

### 3. Development Architecture Follow-up
Only if work packages are not clear.

Output:
- workflow map
- architecture map
- work package map
- resources and timeline

### 4. Quality / Evals Review
Recurring only if active development is underway.

Output:
- test/evals cases
- SME review checklist
- version acceptance criteria
