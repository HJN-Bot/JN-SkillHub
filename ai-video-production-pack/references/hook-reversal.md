# Hook / Reversal

Use for viral openings, AI before/after stories, ordinary-person-to-system-leverage contrast, and more dramatic short-video intros.

Absorbs the useful 12-second density from short-drama skills, but removes genre noise unless the user explicitly wants drama.

## 12s Knowledge/Product Hook Structure

| Time | Job | Pattern |
|---|---|---|
| 0–3s | Stop scroll | surprising question, broken workflow, strong before/after image |
| 3–6s | Reveal stakes | why normal AI chat is not enough |
| 6–9s | Turn | show the system/lifecycle/automation taking over |
| 9–12s | Payoff | one memorable line or visual proof |

## Reversal Patterns

- Scattered → orchestrated
- Chatbot → operating system
- Manual follow-up → autonomous loop
- One-off prompt → reusable skill
- Pretty demo → evidence-backed workflow
- Personal hack → team workflow

## Guardrails

- For company/internal videos, keep drama controlled and credible.
- Use conflict in the workflow, not fake human melodrama.
- Do not add irrelevant short-drama genres like 豪门/修仙 unless requested.
