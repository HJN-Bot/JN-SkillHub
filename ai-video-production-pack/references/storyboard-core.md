# Storyboard Core

Use for any 30s+ or multi-shot AI video.

## Flow

1. Define duration and platform.
2. Split into 4–8s shots. For a 5-min internal sharing video, use major scenes, not every sentence.
3. Lock visual language:
   - palette;
   - lighting;
   - lens / motion language;
   - typography policy;
   - recurring motif.
4. For each shot, define:
   - information job;
   - visual metaphor;
   - composition;
   - camera movement;
   - motion graphics / AI video need;
   - source assets;
   - prompt;
   - negative constraints.
5. Add assembly notes: transitions, subtitles, music/SFX, screenshot insert points.

## Consistency Rules

- Reuse the same 3–5 colors.
- Reuse one motion grammar: slow push, card flow, orbit, timeline snap, or node pulse.
- Use AI video for atmosphere and metaphor; use HTML/Figma/screenshots for exact UI/text.
- Avoid generating readable UI text inside video models; overlay exact text in post.
