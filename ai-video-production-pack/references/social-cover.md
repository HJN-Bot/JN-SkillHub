# Social Cover

Use for Xiaohongshu, Douyin, Bilibili, WeChat Channels, or any high-click cover/thumbnail.

Absorb from Jimeng cover workflows, but keep it adaptable.

## Rules

- One cover = one hook.
- Main title: 2–6 Chinese characters if Chinese cover; short, heavy, readable on mobile.
- Hierarchy: title first, core visual second, background last.
- Translate abstract ideas into visible objects.
- Avoid long text, multiple labels, fake UI text, and generic phrases like “一键生成”.
