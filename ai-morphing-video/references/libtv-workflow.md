# LibTV 详细操作手册 与 OpenClaw 接入指南

## LibTV 基本信息

- 官网：https://www.liblib.tv/
- 定位：一站式 AI 视频创作平台，同时面向人类创作者 + AI Agent
- 背靠：LiblibAI（哩布哩布AI），国内最大 AI 创作社区之一
- 集成模型：可灵O3、可灵3.0、Wan 2.6、Seedance 1.5、Hailuo 2.3、Pixverse V5.5 等 30+

## OpenClaw 接入 LibTV（Skill 包）

### 安装方式

```bash
# 官方 Skill 包（MIT 协议，免费）
# GitHub: github.com/libtv-labs/libtv-skills

# 在 OpenClaw 的 Skill 管理界面中：
# 1. 输入 GitHub 仓库地址
# 2. 获取 LibTV Access Key（官网登录后获取）
# 3. 配置完成后即可用自然语言调用
```

### 典型调用示例

安装 libtv-skills 后，在 OpenClaw/Discord 中可以说：
```
帮我复刻这个视频风格，给我的 [产品名] 做一个宣传片

帮我生成一个爱丁堡旅拍风格的视频，P5红黑风格，5个场景

用这4张照片生成一个30秒的品牌宣传片，风格参考第一张
```

Agent 会自动完成：剧本生成 → 分镜设计 → 图片生成 → 视频生成 → 音频 → 成片链接

## 人工操作：产品广告视频标准流程

以 Sony α7IV 相机案例为例，详细步骤：

### Step 1：创建项目
1. 登录 liblib.tv
2. 点击「开始创作」→ 进入无限画布
3. 新建「图片节点」→ 上传产品照片

### Step 2：优化产品图（可选但推荐）

**打光调整：**
- 点击图片节点右上角「+」→「打光」
- 三维坐标球：拖动光源位置，调整水平/垂直角度
- 颜色调整：科技感用冷蓝光，高端感用暖金光

**多角度生成：**
- 「+」→「多角度」
- 设置俯仰角、方位角，生成产品不同视角
- 适合丰富多镜头素材

### Step 3：生成特效视频

1. 从图片节点拉线 → 新建「视频节点」
2. 模式选「**全能参考**」（这是产品特效的关键！）
3. 模型选「**可灵O3**」
4. 时长：5秒
5. 输入 Prompt（见 prompt-library.md 产品特效模板）
6. 点击生成

### Step 4：多镜头组合

生成 3-5 个不同角度/特效的片段后：
- 在 LibTV 节点图中连线管理所有素材
- 添加「音频节点」→ 输入背景音乐风格描述
- 导出所有片段 → 剪映拼接 → 加字幕/品牌元素

## 人工操作：风格化旅拍标准流程

以爱丁堡 P5 风格案例为例：

### Step 1：在 LibTV 中用图片节点做风格化

LibTV 接入了 Seedream、Midjourney 等图像模型，可以直接做风格化：

1. 上传原始照片
2. 图片节点选「**重绘**」功能
3. Prompt 输入 P5 风格描述
4. 可配合「ControlNet」保留构图（如有）

或者：在 fal.ai 做好风格化图后，导入 LibTV 继续生成视频。

### Step 2：生成 Ken Burns 运镜视频

1. 风格化图 → 拉线到视频节点
2. 模型选可灵O3 或 Wan 2.6
3. Prompt 输入 Ken Burns 运镜描述
4. 为每张图生成一个方向不同的运镜片段

### Step 3：25宫格抽卡选最佳镜头（可选）

如果对单张图有多种运镜想法：
- 使用「25宫格」批量生成不同方向的运镜结果
- 从中挑选最满意的帧作为下一步素材

## 快速参考：可灵O3 vs 其他模型

| 场景 | 推荐模型 | 原因 |
|------|---------|------|
| 产品特效广告 | 可灵O3 | 主体一致性强，支持声画同出 |
| 旅拍运镜 | 可灵O3 或 Wan 2.6 | 运镜流畅，成本可控 |
| 古风/写实人物 | Seedance 1.5 | 中文场景和古风专优 |
| 大批量低成本 | Wan 2.6 | 开源成本最低 |
| 夸张动态 | Pixverse V5.5 | 运动幅度大，适合动感内容 |

## 视频提帧（拉片分析）

LibTV 内置「视频提帧」工具，非常适合：
- 拆解参考视频的镜头语言（景别/运镜/提示词自动生成）
- 找到参考视频的每个关键帧
- 作为自己创作的分镜参考

使用方法：上传参考视频 → 选「视频解析」→ 自动输出分镜表
