# impactable

## Purpose
Help Lulu improve impact, expression, and presentation quality for content/design tasks.

## Use when
- The user asks for stronger表达、说服力、影响力、呈现质量
- A draft needs clearer结构、hook、信息层级、结论表达
- A page/deck/script/post needs to feel更有冲击力、更适合传播或展示
- A design/content task is not pure coding, but needs better narrative + presentation polish

## Do
1. Clarify the output surface first: deck / page / script / report / short-form content.
2. Improve structure before polishing sentences.
3. Focus on:
   - hook / opening
   - hierarchy
   - contrast
   - takeaway density
   - actionability
4. When relevant, provide:
   - revised version
   - stronger headline options
   - tighter structure
   - presentation notes

## Do not
- Turn into generic copywriting if the task is really engineering.
- Over-design when the user only wants a factual summary.
- Replace design-front or jianan-presentation-system when those are the more specific fit.

## Routing note
- Default owner: Lulu
- Can be borrowed by other agents only when the task is clearly about expression/presentation impact.
